# Mi Proyecto DevOps

Este es un proyecto de práctica para aprender el flujo de trabajo entre **GitHub**, **AWS S3** y el navegador, como parte de mis estudios de DevOps.

## 📂 Estructura
- `src/index.html`: Página principal
- `src/styles.css`: Estilos del sitio
- `README.md`: Documentación del proyecto
- `.gitignore`: Archivos a ignorar en Git

## 🚀 Objetivo
- Practicar despliegue en AWS S3 con hosting estático.
- Usar GitHub para versionar el código.
- Mejorar mis habilidades en DevOps.
